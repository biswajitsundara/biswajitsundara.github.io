About Jalam Framework


Jalam is an UI automation Framework that comes with Apache licence. This is free and will always be. If you like the work and want to support then please check the support section for details.


UI automation has been a challenge across the industry, if we take a look, there will be millions, billions of flaky UI test scripts on GitHub. There are various factors those cause the unstable test automation scripts. This will be a really long explanation if I talk about those factors however If I pick few of the most pressing issues then those will be technical debt, incorrect automation strategy. So eventually the automation project slips and after a while it becomes really difficult to bring it back. Here with Jalam Framework I am trying to solve the problem and help the teams to focus on the test scripts without worrying much about the underlying framework.




Vision


Our vision is to offer a framework to the QA community to develop better and efficient automation scripts. Currently there are options available in the market however those are either paid or scriptless automation. Jalam Framework is free to use and it allows the users to code. Also the most important thing about this framework is simplicity. We want this framework to be the most efficient and at the same time retain its simplicity.
