VB Script Basics

VB Script is a light weight scripting language developed by Microsoft. It is used to manage windows system as well as client side scripting.

  • VB script was released by Microsoft in 1996
  • It uses Component Object Model to access elements in the executing environment.
  • By default VB script is installed in Windows Operating system.
  • Simply put some code in a notepad and save it with .vbs extension, then double click on the file to execute the code.
  • Some of most popular IDEs for VB Script are - vbs edit, visual studio
  • VB script is quite popular with UFT automation tool.
  • We can use VB script in web pages also however, it works only in IE browser.
  • Debugging VB script is bit difficult
  • System Admins use VB scripts to code system automation utils
  • VB script is used in Excel Macros and can any company exist without Excel!
  • To comment any line use single quote. Example: 'This is a comment

Open NotePad and type the below code. Then save it with file name demo.vbs and then double click on the file to execute

Data Types & Variables

To keep it simple, we can say variable is the container where we store our data and data type is the specifier for the variable, means using data type we will declare which type of data a particular variable is going to contain.

We can take real world example of our kitchen where we have different containers for oil,sugar, biscuit etc. And when we purchase those containers from market, they specify which container is for which item and when we bring it home, we keep changing the item that we store in the containers. That's exactly how the data type, variable concept works. Also we know there are small containers, big containers. Similarly we have data types with small storage capability and big storage capability.

  • Variable name must begin with a letter. Example: name, age, sal
  • Variable name can't start with special letters, numbers. Example: 1name,$age, sal. etc
  • This is the syntax to declare a variable: Dim name
  • We can't declare and assign the variable at the same time. Dim name="Biswajit"
  • First we need to declare Dim name and then assign the value.name="Biswajit"
  • We can also use variables directly without declaring, however it's not the best practice.name="Biswajit"
  • VB Script has an option to force you to declare variables. Use the statement Option Explicit at the top of your script
  • We don't have to specify the data types like integer, string etc. Based on the value assigned to the variable, it understands the data type.
  • We can declare multiple variables separated by comma. Dim name,age,sal
  • If we want to know the type of variable, then we can use this method typeName(variablename). It will return the data type as Integer, String etc.
  • The scope of variables can be Dim (local to procedure, script etc), private (if we declare it globally in one script then it will be available for all the functions in the same script), public (if we declare it globally in one script then it will be available for all the associated scripts also)

Control Flow Statements

The VB Script statements are executed sequentially always means the line number1 will be executed first and then line number2 and so on. However when we implement conditions, decision making blocks then we need to execute statements based on the logic that it satisfies.

So this is called control flow mechanism where we can control the execution order of statements.

  • Control flow statements available in VB Script are: If Else, Loops, Select Case
  • To break control flow statements we have these commands: Exit For, Exit do, Exit Sub, Exit Function
  • There's no command to break select case or if.
  • Return command is not applicable in VB Script also

If Else

X=inputbox("Input value for X")

Y=inputbox("Input value for Y")

If (X>Y) Then

    msgbox "Value of X is greater than Y"
    
    ElseIf (Y>X) Then
        msgbox "Value of Y is greater than X"
    
    Else
        msgbox "X and Y are equal"

End If

Arrays

Array is a collection of similar items or we can say when we need to store a series of data, we can go for array.
  • A variable contains a single value, Array can contain multiple values.
  • Items in array are stored in a sequence, refered by index 0,1,2..etc
  • Array size/dimension specifies what is the maximum elements it can store.
  • Array index starts from Zero (0). If the size is 5 then it can hold upto 6 elements.
  • Array index can't be negative.
  • Generally array is meant to contain only homogeneous data. Example an array can have only integer or string type data, we can't store both type of data in one array.
  • VB Script array can contain any type of data.
